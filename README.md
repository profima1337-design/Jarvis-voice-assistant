# Jarvis Voice Assistant

Локальный голосовой ассистент на Python для Windows с распознаванием речи на основе Vosk.

## О проекте

Этот проект реализует простой голосовой помощник "Jarvis", который слушает микрофон, определяет команду и выполняет действия:
- открытие сайта или браузера;
- запуск и закрытие приложений;
- управление звуком и окнами;
- воспроизведение звуковых ответов;
- расширяемость через GUI API.

## Структура проекта

- `voice_input.py` — основной скрипт.
- `requirements.txt` — список зависимостей Python.
- `model/` — папка с Vosk-моделью для распознавания речи.
- `sounds/` — звуковые файлы для ответов.


## Модель распознавания речи

Для работы Jarvis требуется папка `model/` с Vosk-моделью. В проекте уже есть русская модель, но вы можете заменить её на другую совместимую модель Vosk.

> Если папка `model/` отсутствует, программа завершится с ошибкой.

## Запуск

```powershell
python voice_input.py
```

После запуска ассистент слушает микрофон и активируется по одним из слов:
- `джарвис`
- `джарвиз`
- `чарвис`
- `жарвис`
- `гарвис`
- `арвис`
- `джарвисе`

После активации голосом произнесите команду.

## Примеры команд

- `джарвис, найди в браузере погоду`
- `джарвис, открой youtube`
- `джарвис, включи музыку`
- `джарвис, открой телеграм`
- `джарвис, закрой браузер`
- `джарвис, сделай громче`
- `джарвис, сверни все окна`

## Доступные команды

### Общие фразы
- `привет`, `здравствуй`
- `как дела`, `как настроение`
- `спасибо`, `благодарю`

### Интернет и браузер
- `найди`, `загугли`, `что такое`, `почему`
- `музыку`, `youtube`, `яндекс`, `гугл`
- `браузер`, `интернет`, `сайт`

### Приложения
- `стим`, `роблокс`, `телеграм`, `дискорд`
- закрытие через `закрой`, `выключи`

### Системные команды
- `диспетчер задач`
- `окна`, `вкладки`, `все`
- `окно`, `сверни`, `закрой`
- `звук`, `громче`, `тише`, `муте`

## GUI API

Можно добавлять команды из внешнего GUI-модуля через функцию:

```python
voice_input.add_gui_command(["включи стрим"], "web_page", "https://twitch.tv")
```

Поддерживаемые типы действий:
- `sound`
- `web_page`
- `run_script`
- `function`

## Зависимости

Список зависимостей в `requirements.txt`:

- `pyautogui`
- `pygame`
- `sounddevice`
- `vosk`

## Важные замечания

- Проект рассчитан на Windows.
- Убедитесь, что работает микрофон и звуковая карта.
- При проблемах с воспроизведением звука проверьте настройки `pygame`.
- `pyautogui.FAILSAFE` включён: перемещение мыши в угол экрана остановит скрипт.

## Лицензия

Проект предоставлен без явной лицензии и используется для локального эксперимента.
